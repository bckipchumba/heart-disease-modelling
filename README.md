# Heart Disease Prediction Using Logistic Regression

## Overview
This project applies statistical modelling and machine learning techniques to analyse factors associated with heart disease risk using the UCI Heart Disease dataset.


## Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Statsmodels
- Matplotlib



## Techniques Used
- Data cleaning and preprocessing
- Exploratory data analysis
- Logistic regression
- Predictive modelling
- Model evaluation

## Objectives
To investigate relationships between clinical variables and heart disease outcomes and develop predictive analytical models.
